# Place your background music MP3 file here

## Instructions:

1. Name your file: `background-music.mp3`
2. Place it in this folder: `/app/frontend/public/music/`
3. Go to admin panel: Site Settings â†’ Background Music Settings
4. Toggle "Enable Background Music" to ON
5. Click Save
6. Your music will play on the site!

## Recommended:
- File format: MP3
- File size: Under 5MB
- Type: Instrumental/ambient music
- Volume: Set to 0.2-0.4 in admin panel for subtle background
