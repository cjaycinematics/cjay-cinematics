## ğŸ“ What You Can Edit

### **Pricing Packages**
- Click "Pricing Packages" in the sidebar
- Click any package to edit price, features, etc.
- Click "New Pricing Package" to add more options

**Edit fields:**
- Package Name (e.g., "3-Minute Cinematic")
- Price (e.g., "$125")
- Description (e.g., "Perfect for highlights")
- Features (add as many as you want)
- Featured (toggle for special badge)

### **Testimonials**
- Click "Testimonials"
- Click any testimonial to edit
- Click "New Testimonial" to add customer reviews

**Edit fields:**
- Client Quote (the testimonial text)
- Client Name
- Service Type (e.g., "Memorial Service")

### **Founder Bio**
- Click "Founder Information"
- Update your name, title, biography
- Add or remove photos
- Each paragraph is separate for easy editing

### **Portfolio**
- Click "Portfolio"
- Add videos or photos of your work
- Click "New Portfolio" to add items

**Edit fields:**
- Title (e.g., "Smith Family Memorial")
- Category (Memorial, Celebration, Events)
- Type (Video or Photo)
- Video/Image URL (YouTube embed link or image URL)
- Description (optional)

### **Site Settings**
- Click "Site Settings"
- Update business info, email, social links
- Change tagline or service area
